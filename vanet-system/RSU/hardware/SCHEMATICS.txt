# RSU Wiring Schematic (Text)

Power:
- 12V input -> Buck 5V -> 3.3V LDO for ESP32 and LoRa
- Dedicated 4.0V regulator for SIM800L
- TVS diode and resettable fuse on 12V input

ESP32 <-> LoRa (SX127x):
- SPI: SCK (GPIO18), MOSI (GPIO23), MISO (GPIO19), NSS (GPIO5)
- DIO0 -> GPIO26 (interrupt)

ESP32 <-> SIM800L:
- UART2 TX (GPIO17) -> SIM800L RX (via level shifter if required)
- UART2 RX (GPIO16) <- SIM800L TX
- SIM800L PWRKEY control -> GPIO4 (to control power on/off)
- RI (Ring Indicator) -> optional GPIO for incoming call/SMS

microSD:
- SPI or SDIO interface to ESP32 (use 3.3V level)

Antennas:
- SMA connectors for LoRa and GSM.

Ensure RF separation between LoRa and GSM antennas to reduce interference.
